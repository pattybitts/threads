Welcome to Threads of the Pattern! Enter a command below.

Valid commands:
add_char=<Character Name>: Presents form to add new character to library.
edit_char=<Character Name>: Presents form to edit character info in library. Accepts aliases.
disp_save: Displays save file info.
disp_book: Displays database info for book in save file.
disp_chapter=<Chapter Name>: Displays database info on named chapter for book in save file.
disp_char=<Character Name>: Displays databse info for named character. Accepts aliases.
text_tool: Opens text parsing tool for current save file.
